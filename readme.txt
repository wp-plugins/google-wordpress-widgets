== Description ==

Display the latest post from your Google+ Plus accounts inside WordPress widgets. Included QR code for Google+ Profile.

Customize the number of post displayed and user allowed to include updates from Google Buzz too.

See the [Google+ Plus Wordpress Widgets](http://patrick.bloggles.info/) for further information.

[Live Demo](http://patrick.bloggles.info/)

== Installation ==

1. Upload `google-plus-wordpress-widget.php` to your `/wp-content/plugins/ directory`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar from `Appearance->Widgets` and configure the widget options.
4. You can try scanning this code from your phone right now.(It will take you to your Google+ Page)


== Frequently Asked Questions ==

= Can multiple instances of the widget be used? =

Yes.

= Can private Google+/Buzz accounts be used? =

No. The widget does not support authenticated requests for private data.

= Can I just show profile with button? =

Yes, checked the option `Hide All Updates` and `Hide Work Info`.
